Wishlist Demo REST API Service

GET /wishlists to see all the wishlists.
GET /wishlists/{id} retrieves a wishlist from the DB using an ID.
PUT /wishlists/{id} updates a wishlist using an ID.
DELETE /wishlists/{id} removes a wishlist from the DB using an ID.
POST /wishlists creates a new wishlist.

Example JSON Data:

{"name": "My First Wishlist", "id": "1, "account_id": "234", "items": "PM Job"}
